Este é um portfólio pessoal, desenvolvido usando HTML, CSS, JavaScript e PHP com auxílio do framework Bootstrap.
Contém os seguintes menus:
Sobre: possui um breve resumo sobre a minha história na tecnologia e também algumas curiosidades sobre mim;
CV: contém dados mais detalhados sobre a minha educação, experiência, participações e méritos;
Portfólio: contém um conjunto de projectos desenvolvidos e também que ainda estão sendo desenvolvidos;
Contacto: contém todos dados necessários para contacto em caso de interesse pelos serviços.
